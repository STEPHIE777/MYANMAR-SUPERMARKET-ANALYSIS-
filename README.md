# MYANMAR-SUPERMARKET-ANALYSIS-
THESE ANALYSIS CAPTURES DETAILED TRANSACTION RECORD FROM WINMART SUPER MARKET  LOCATED IN MYANMAR, AND HAS BRANCHES ACROSS THREE CITIES: YANGON, NAYPYITAW AND MANDALAY.  IT SHOWS A COMPREHENSIVE VIEW OF SALES ACTIVITIES, CUSTOMER DEMOGRAPHICS AND PAYMENT METHOD FROM JANUARY TO MARCH 2019.
